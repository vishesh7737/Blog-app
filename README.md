# Blog-app
 A full-stack Blog Web Application built with HTML, CSS, JavaScript, Node.js, and Express. It allows users to create and view posts through a clean interface. The project demonstrates API handling, server management, and cloud deployment, and is successfully hosted on Render for public access.
Key Features
Create and publish blog posts

View all posts dynamically

Smooth navigation using hash routes (#posts)

Responsive and clean user interface

Backend API handling requests and data

Cloud deployment for global access

Real-time interaction between frontend and backend

Technologies Used
Frontend

HTML

CSS

JavaScript

Backend

Node.js

Express.js

Tools & Deployment

Git & GitHub for version control

Cloud hosting using Render

Project Objective
Learn full-stack web development workflow

Understand server creation and API handling

Gain hands-on experience with cloud deployment

Build a real-world production-ready web application

Outcome
The application was successfully deployed and is accessible through a live URL. Users can access and interact with the blog without running it locally.
